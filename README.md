# Tracking US Foreign Policy Interests
>Eric Gutiérrez, 10th February 2026.

### Visit the interactive [dashboard](https://ericgutierrezm.github.io/us_foreign_policy_interests/)

This project applies **Natural Language Processing** (NLP) to 700+ expert reports from the *Council on Foreign Relations*. It tracks the salience of 200 key topics, revealing how US strategic priorities have shifted.

### Project Overview
* **Data Source:** 700 expert reports (1998–2025).
* **Methodology:** Natural Language Processing (Lemmatization + DTM).
* **Visualization:** Interactive time-series analysis using Chart.js.
